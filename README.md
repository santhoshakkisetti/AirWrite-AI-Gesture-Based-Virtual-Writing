# AirWrite AI – Week 2

## Overview

This repository contains the project I built during **Week 2** of my AI learning journey.

The focus was on exploring **Model Context Protocol (MCP)**, AI integrations, computer vision, and building a real-time **AirWrite AI** application that converts hand gestures into digital text using AI-powered hand tracking.

---

## Project

### AirWrite AI

Built an AI-powered air writing application that tracks finger movements through a webcam and converts gestures into digital text in real time.

### Features

* Real-time hand tracking
* Air writing using finger gestures
* Gesture-to-text conversion
* Interactive virtual notepad
* AI-powered computer vision
* Smooth and responsive user interface
* Webcam-based interaction
